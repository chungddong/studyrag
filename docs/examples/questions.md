# SMART 학습공동체 OT 예상 질문

아래 질문들은 OT 자료(ot_raw.txt)를 기반으로 챗봇이 잘 답하는지 확인하기 위한 테스트 질문입니다.

## 일정/장소
- 학습기간은 언제부터 언제까지야?
- 학습은 총 몇 주 / 최소 몇 회 해야 해?
- 중간고사 기간에는 학습을 쉬어야 해?

## 참여/신청
- 신청 기간은 언제야?
- 신청서는 누가 제출하고 어디로 제출해?
- 팀 구성은 최소/최대 몇 명이야?
- 전공 제한이 있어? 어떤 주제로 해도 돼?

## 활동/의무사항
- 주 1회 몇 시간 이상 학습해야 해?
- 성찰일지는 언제까지 제출해야 해?
- 성찰일지 제출을 안 하면 어떤 불이익이 있어?
- 학습코칭은 총 몇 번이고, 각 코칭은 어떤 내용이야?
- 학습코칭은 대면이야? 줌으로 해도 돼?

## 사전/사후 검사 / 시청 과제
- 사전검사는 언제까지 해야 해?
- 만족도/핵심역량 검사는 어디서 해?
- ‘선후배에게 듣는 학습전략-공동체편’ 시청은 언제까지야?
- 배움맛집 동영상 강의는 필수야? 가산점이 있어?

## 지원금/정산
- 학습지원금은 얼마고 언제 입금돼?
- 지원금은 언제부터 사용한 걸로 인정돼?
- 줌으로 하면 다과/식비 지원이 가능한가?
- 식비/다과는 1인당 얼마까지 지원돼?
- 지원금으로 어떤 항목(도서/앱/응시료/문구류/복사 등)을 쓸 수 있어?
- 영수증 증빙은 어떤 게 필요해?
- 학습 중단하면 지원금은 어떻게 돼?

## 계획 변경/결과보고서
- 예산 변경은 어떻게 요청해?
- 결과보고서는 언제까지 어디에 제출해?

## 우수팀/평가
- 우수팀은 몇 팀이고 상금/장학금은 얼마야?
- 우수팀 평가는 어떤 항목으로 점수 계산해?

## 문의
- 문의처(장소/센터/연락처)는 어디야?
